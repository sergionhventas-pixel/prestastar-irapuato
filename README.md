# PrestaStar Promotor Suc. Irapuato - Sitio Web Comercial

## 📋 Descripción

Sitio web comercial moderno y profesional para **PrestaStar Promotor Suc. Irapuato**, una financiera mexicana especializada en la distribución de vales de dinero y oportunidades de negocio para emprendedores.

## 🎯 Características Principales

### Página de Inicio (Home)
- **Sección Hero:** Mensaje principal atractivo con llamadas a la acción
- **Botones CTA:** "Solicita Información" y "Contactar por WhatsApp"
- **Animación Visual:** Estrella flotante que representa la marca

### Sección Sobre Nosotros
- **Descripción del Negocio:** Explicación clara de qué es PrestaStar
- **Tarjetas de Características:** Préstamos Rápidos, Comisiones Altas, Respaldo Sólido, Flexibilidad
- **Beneficios de Ser Distribuidor:** Lista completa de ventajas
- **Diseño Responsivo:** Adaptado a todos los dispositivos

### Galería de Promociones
- **6 Tarjetas de Contenido:** Promociones, Vales, Rifas, App, Comunidad, Éxito
- **Efectos Hover:** Animaciones al pasar el mouse
- **Diseño Moderno:** Gradientes y colores atractivos

### Sección Cómo Funciona
- **4 Pasos Claros:** Regístrate → Capacitación → Distribuye → Cobra
- **Numeración Visual:** Círculos con números para mejor comprensión
- **Descripción Detallada:** Cada paso tiene su explicación

### Sección de Contacto
- **Información de Contacto:** Ubicación, WhatsApp, Redes Sociales
- **Formulario de Contacto:** Campos para Nombre, Email, Teléfono, Mensaje
- **Integración WhatsApp:** El formulario envía mensajes directamente a WhatsApp
- **Enlaces a Redes Sociales:** Facebook, Instagram, TikTok, YouTube

## 🔧 Cómo Usar

### Localmente
```bash
# Navegar al directorio
cd prestastar_irapuato_web

# Iniciar servidor local (Python 3)
python3 -m http.server 8000

# Abrir en navegador
# http://localhost:8000
```

### Desplegar en Línea
Puedes subir los archivos a cualquier servicio de hosting:
- **GitHub Pages** (gratuito)
- **Netlify** (gratuito con dominio personalizado)
- **Vercel** (gratuito)
- **Hosting compartido** (con cPanel)

## 📝 Personalización

### Cambiar Colores
Edita las variables CSS en `styles.css`:
```css
:root {
    --primary-color: #6B46C1;
    --secondary-color: #EC4899;
    --accent-color: #10B981;
}
```

### Actualizar Información de Contacto
Edita el archivo `index.html` en la sección de contacto:
- Número de WhatsApp
- Ubicación
- Enlaces a redes sociales

### Agregar Más Contenido
- Galería: Agrega más tarjetas en la sección `gallery-grid`
- Beneficios: Añade más items en `benefits-list`
- Pasos: Crea más elementos en la sección `steps`
